# Security

Report vulnerabilities by opening a private security advisory on GitHub or contacting me directly.

When possible, include:

- A short description of the issue and why it matters.
- Affected version(s)
- Reproduction steps or a small proof of concept.
- Any suggested fix or mitigation.

I will review reports on a best-effort basis and coordinate a fix before any public disclosure.

Please do not disclose unresolved vulnerabilities publicly (i.e. via an issue) until a fix is available.
